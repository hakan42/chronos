chronos - Χρόνος - greek god of time

a LED clock controller with REST API

What? Why? And Wherefore?

My wall clock died. And, instead of buying a new one like any sensible person would do, I started creating my own LED clock. And being the computer geek that I am, I wanted it to be controllable by my home automation system.

Hardware

The clock consists mainly of:

  • A Raspberry Pi, bought from Amazon
  • A ring with 60 LEDs on it, bought from watterott electronics
  • A FadeCandy LED controller, bought from Amazon

Software

  • A Spring Boot based Java application
  • The native controller for the FadeCandy
  • A dockerified boot image for the Pi, thanks to the guys at Hypriot
